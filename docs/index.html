Probability Engine

Monte Carlo Simulation

Every tweet includes a real-time Monte Carlo probability — a statistical technique that runs 10,000 simulated versions of the remaining game to estimate the odds of completion. Each simulation uses live pitcher and opponent data, so the number updates after every single out.
